Home Server

Description: Built my own server using computer parts that I had bought. Currenly it is running UbuntuServer with a SambaShare for computer backups and it is hosting a VM that runs my modded Mineraft Server.

Hardware: CPU Ryzen 5 5600X, RAM: 16Gb 3200Mhz, 1TB Nvme SSd

Key Contributions

  1. Got hands on with computer networking
    Running a Minecraft server taught me alot about computer networking.
  2. Setup a Virtual Machine on Ubuntu Server
    Setting up a VM on Ubuntu Server was challenging. I used KVM/QEMU to create a VM running Ubuntu Server to host my Minecraft server, but I encountered issues accessing the console of the VM within the host OS. To resolve this, I installed XLaunch to use X11 with Windows 10, which provided access to the GUI environment on the Ubuntu Server guest VM. Since Ubuntu Server is command-line only, this was necessary. I then used PuTTY to SSH into the VM and access the GUI on my Windows 10 PC.
  3. After all that, I was able to configure the guest OS and install the software needed to host my modded Minecraft server. I used Playit.gg to create a VPN tunnel, connecting my server to their network, providing a seamless LAN-like experience for users. This was a fantastic project! I learned a lot about networking, especially troubleshooting, which was a significant part of the early setup. PI-Hole Network Ad Blocking System

Description: Implemented a network wide ad-blocking and DNS filtering solution using Pi-hole on a Raspberry Pi to my home network.

Technology used:
Raspberry Pi 4b, Linux(Ubuntu Server),Pi-hole,Network Manager

Key Contributions

  1. Configured and deployed Pi-hole to filter ads and malicious domains for all devices on the network.
  2. Optimized performance by fine-tuning blocklists and DNS settings for efficient ad-blocking without affecting browsing speed.
  3. Enhanced network security by identifying and blocking trackers and unwanted traffic.
  4. Integrated Pi-hole with existing home network infrastructure, including dynamic IP configuration and custom DNS settings.

Outcome: Reduced data usage, improved network performance and security.

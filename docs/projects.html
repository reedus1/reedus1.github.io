Raspberry PI DNS Server and AdBlocker

What is Pi Hole?

Pi-hole, Created by Jacob Salmela in 2014, is a network-wide ad blocker that acts as a DNS-based sinkhole. It prevents ads, trackers, and malicious domains from loading across all devices connected to a network. Pi-hole does this by comparing DNS queries against a blacklist of known ad and tracking domains. If a requested domain matches an entry in the blacklist, Pi-hole responds with a null IP address (e.g., 0.0.0.0 or 127.0.0.1), effectively blocking the domain.

What I learned From this Project

During this project, I learned a lot about using Ubuntu Server on my Raspberry Pi (RPI for short). I set up a DNS server on my RPI and then figured out how to communicate with it from my other devices. The process was straightforward: I configured my router to use the RPI as the primary DNS server instead of the default ISP-provided DNS.
